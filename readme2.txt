./a.out config{4,5}.txt                //runs the simulation with the given config file

The available config files are config4.txt and config5.txt for simulation of added features

Use config{1,2,3}.txt for older simulations

For Vehicles the syantax in config file is 
Buses=COLOR
bike=COLOR
Truck=COLOR
Car=COLOR
Auto=COLOR

The available colors are RED,YELLOW,GREEN,BLUE,WHITE

The number of lanes is fixed at four
lane_width=road_width/4
lane_width>=4
